Create a modern, fully responsive NGO website UI/UX design for an organization named “Better Future for Pakistan”. The design style should follow clean Minimalism with a premium educational-tech feel, soft typography, lots of whitespace, subtle animations, smooth transitions, and a professional color palette inspired by Pakistan (green, white, soft emerald, light gray accents). The interface should feel trustworthy, inspiring, modern, and easy to use for teachers across desktop, tablet, and mobile devices.
The platform is a Teacher Learning Management System (TLMS) where administrators manage teachers, courses, exams, and certifications.
Design Requirements:
1.	Overall Style
•	Minimalistic modern layout
•	Fully responsive for mobile, tablet, laptop, and ultra-wide screens
•	Elegant typography and accessibility-focused UI
•	Smooth hover effects and micro-interactions
•	Soft shadows, glassmorphism/light cards where suitable
•	Professional NGO + EdTech aesthetic
•	Clean dashboard layouts with proper spacing
•	Animated statistics counters and subtle scroll animations
•	Dark/light mode support
•	High-quality icons and illustrations
•	Premium UI similar to modern SaaS platforms
2.	Landing Page Sections
•	Hero section with inspiring headline:
“Empowering Teachers for a Better Future in Pakistan”
•	NGO mission and impact section
•	Teacher training overview
•	Featured learning programs
•	Certification benefits
•	Statistics section:
o	Teachers trained
o	Courses completed
o	Certificates issued
•	Testimonials from teachers
•	Partners/supporters section
•	Contact & support section
•	Footer with quick links and social icons
3.	Authentication System
Design modern authentication pages:
•	Admin Login
•	Teacher Login
•	Forgot Password
•	Secure OTP verification UI
•	Minimal and elegant forms
•	Animated login illustration

Create a powerful admin panel with sidebar navigation and analytics cards.
Admin Features:
